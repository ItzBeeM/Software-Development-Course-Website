Lesson 2: IDE

In this lesson we talk about IDEs (Integrated Development Environments) and their importance in software development. We will cover popular IDEs like Visual Studio Code, Vim and discuss their features and benefits for developers.

You technically could write code in a notepad, but using an IDE will be a much more pleasant experience. One of the most popular IDEs is Visual Studio Code It offers a wide range of features, including syntax highlighting, code completion, debugging tools, and a large library of extensions that can enhance its functionality. On the other hand we have vim which is scary when you first look at it, but once you get used to it, It's a powerful tool for editing code. It works mainly through keyboard shortcuts, so when you master it is much faster and convenient.
